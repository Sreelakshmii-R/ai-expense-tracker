# AI-Powered Smart Expense Tracker

A web application to track and categorize personal expenses using AI.

## Features

- Add and delete individual expenses
- AI-based automatic expense categorization (Python Flask)
- Dynamic chart to visualize expenses (Chart.js)
- Responsive and user-friendly frontend (HTML, CSS, JavaScript)
- Backend with Node.js server
- Delete individual expenses only (no full reset)

## Tech Stack

- **Frontend:** HTML, CSS, JavaScript, Chart.js
- **Backend:** Node.js, Express.js
- **AI Service:** Python Flask for expense category prediction
- **Version Control:** Git & GitHub
